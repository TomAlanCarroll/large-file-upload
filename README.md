# large-file-upload
Large file upload servlet

This servlet handles large files (up to 10GB) uploading via an HTML5-based JSP.

The following screenshot shows the servlet handling up to a 10GB file from Chrome:
![alt tag](https://raw.githubusercontent.com/TomAlanCarroll/large-file-upload/master/results.png)

The upload limits for each major browser can be found here:
http://www.motobit.com/help/scptutl/pa98.htm
